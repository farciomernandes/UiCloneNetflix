## Rodando

<p> A seguir estão as instruções que devem ser seguidas para executar o sistema em sua máquina. </p>


## No diretório src

1. Instalar as depências
    ```
    yarn
    ```
2. Rodar os testes
    ```
    yarn test
    ```
3. Gerar o coverage dos testes
    ```
    yarn test:coverage
    ```
4. Executar o cliente
    ```
    yarn start
    ```
